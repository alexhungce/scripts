**Python:**
* Write clean, modern Python 3; follow PEP 8 (4-space indentation, 88-column line length).
* Assume `uv` is used for virtual environment and package management.
* Use type hints for function signatures.
* Structure scripts so they execute cleanly inside `uv` virtual environments when called from bash.
